Proximity Query (PQ)

Proximity Query (PQ) is a process to calculate the relative placement of objects. It is a critical task for many applications such as robot motion planning, particularly those involving human-robot collaborative control. This program computes a PQ formulation which can support non-convex objects represented by meshes or cloud points.
